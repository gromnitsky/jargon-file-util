How FTS works

Selecting the FTS option downloads an index. After that, you can search using:

Query Desc
foo bar foo OR bar
+foo +bar foo AND bar
+foo -bar foo AND NOT bar
foo* match words prefixed with foo

You can't share a link to FTS results, as it would require the user to wait for a potentially large index to download before being able to read anything.

Internally, FTS is power by Lunr.js. No server-side code is used.
